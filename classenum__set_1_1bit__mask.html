enum_set::bit_mask<Size>::bit_mask(Bools... bits) constexpr noexcept

Constructs a bit mask from a set of boolean bits... specifying which bits to set. The number of bools must match the Size of the bit mask.

template<size_t Size> template<typename... Indices, bool Enable = ! detail::any(std::is_same<Indices, bool>::value...), std::enable_if_t<Enable, nullptr_t> = nullptr>
enum_set::bit_mask<Size>::bit_mask(Indices... indices) constexpr

Constructs a bit mask from a set of indices... specifying which bits to set. The Indices... must be implicitly convertible to size_t, and none should be a bool (that is reserved for bit_mask(Bools...) constructor). Specifying any invalid index throws an std::out_of_range exception.

Test of a bit at a specified index. The index must be less than the bit mask size, otherwise an exception is thrown. Returns true if the bit at index is set, otherwise false.
